THE ATTACK TYPE IS BENIGN

Normal, safe traffic with no malicious activity detected. Depends on traffic patterns.

{% elif prediction == 1 %}

THE ATTACK TYPE IS BOT

Automated malicious scripts that target systems. Depends on request frequency, user agents, and behavior patterns.

Precaution: Use CAPTCHA, monitor user behavior, block suspicious IPs, and analyze traffic for anomalies.

{% elif prediction == 2 %}

THE ATTACK TYPE IS BRUTE FORCE

Attempts to guess credentials by repeatedly trying combinations. Depends on login attempts and response times.

Precaution: Implement account lockout policies, enable multi-factor authentication, and monitor login attempts.

{% elif prediction == 3 %}

THE ATTACK TYPE IS DDOS

Overloads systems with massive traffic. Depends on request volume and source diversity.

Precaution: Use load balancers, enable DDoS protection services, and monitor traffic patterns in real-time.

{% elif prediction == 4 %}

THE ATTACK TYPE IS DOS GOLDENEYE

Floods servers with GET requests to overwhelm resources. Depends on request headers and traffic size.

Precaution: Use rate limiting, enable web application firewalls, and analyze incoming traffic for abnormalities.

{% elif prediction == 5 %}

THE ATTACK TYPE IS DOS HULK

Generates high POST request volumes to exhaust resources. Depends on payload size and request frequency.

Precaution: Monitor POST requests, implement rate limiting, and use traffic filtering tools.

{% elif prediction == 6 %}

THE ATTACK TYPE IS DOS SLOWHTTPTEST

Sends incomplete HTTP requests to tie up server connections. Depends on connection timeouts and headers.

Precaution: Adjust server timeout settings, use application-layer firewalls, and monitor slow connections.

{% elif prediction == 7 %}

THE ATTACK TYPE IS DOS SLOWLORIS

Keeps connections open to exhaust server resources. Depends on incomplete HTTP headers and connection time.

Precaution: Limit the number of simultaneous connections and enable idle connection timeouts.

{% elif prediction == 8 %}

THE ATTACK TYPE IS FTP-PATATOR

Brute force attacks targeting FTP credentials. Depends on login attempts and FTP response times.

Precaution: Use strong FTP credentials, monitor failed login attempts, and enable IP blocking.

{% elif prediction == 9 %}

THE ATTACK TYPE IS HEARTBLEED

Exploits OpenSSL vulnerabilities to access sensitive data. Depends on malformed heartbeat requests.

Precaution: Update OpenSSL libraries, patch vulnerabilities, and use secure configurations.

{% elif prediction == 10 %}

THE ATTACK TYPE IS INFILTRATION

Stealthy unauthorized access to internal systems. Depends on unusual data exfiltration behavior.

Precaution: Monitor internal traffic, use intrusion detection systems, and limit access rights.

{% elif prediction == 11 %}

THE ATTACK TYPE IS PORTSCAN

Scans open ports to identify vulnerabilities. Depends on connection attempts and response patterns.

Precaution: Use port filtering, monitor unusual connection patterns, and disable unnecessary services.

{% elif prediction == 12 %}

THE ATTACK TYPE IS SQL INJECTION

Injects malicious SQL queries into input fields. Depends on input validation and query patterns.

Precaution: Sanitize user inputs, use parameterized queries, and deploy web application firewalls.

{% elif prediction == 13 %}

THE ATTACK TYPE IS SSH-PATATOR

Brute force attacks on SSH services. Depends on login attempts and authentication failures.

Precaution: Use strong SSH keys, enable multi-factor authentication, and monitor login attempts.

{% elif prediction == 14 %}

THE ATTACK TYPE IS XSS

Injects malicious scripts into web input fields. Depends on input validation and script execution behavior.

Precaution: Sanitize and validate inputs, use Content Security Policies (CSP), and escape HTML output.
